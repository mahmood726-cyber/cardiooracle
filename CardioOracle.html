How it works
  1. Enter a valid NCT ID (format: NCT followed by 8 digits).
  2. The engine retrieves trial metadata from the embedded training corpus.
  3. Similarity scores are computed across drug class, endpoint type, comparator, population, and era.
  4. A Bayesian ensemble (Bayesian prior + conditional power + meta-regression) produces a probability estimate.
  5. Output includes a traffic-light confidence band and the top similar trials used as evidence.

CardioOracle v1.0.0
Cardiovascular Trial Outcome Predictor

A hybrid Bayesian + ML tool for predicting the probability that a cardiovascular clinical trial will meet its primary endpoint.

Training data: 784 Phase 2/3 and Phase 3 CV trials from AACT/ClinicalTrials.gov
Model: Bayesian historical borrowing + conditional power + logistic meta-regression
Temporal validation AUC: 0.745  |  Brier: 0.197

Citation:

[Author]. CardioOracle: An Open-Access Tool for Predicting Cardiovascular Trial Outcomes Using Bayesian Historical Borrowing and Design Feature Analysis. 2026. Available at: [URL]

Data source: ClinicalTrials.gov (AACT database)
License: Open access

Methodology

Training Data

784 Phase 2/3 and Phase 3 cardiovascular trials extracted from the AACT database (ClinicalTrials.gov). Trials must be randomized, interventional, enrollment ≥50, with posted results. Success/failure labels assigned via 3-tier system: automated p-value extraction (Tier 1, 97.6%), CI-based heuristics (Tier 2), and manual curation of 29 landmark trials (Tier 3).

Prediction Model

Ensemble of three components:

  • Bayesian Historical Borrowing (40%): Beta-Binomial posterior from K most similar completed trials, weighted by 5-dimension similarity (drug class, endpoint, comparator, population, era).
  • Conditional Power (35%): Schoenfeld formula for time-to-event; arcsine for binary; SMD for continuous endpoints. Uses historical effect size distribution.
  • Design Feature Meta-Regression (25%): L2-regularized logistic regression on 18 trial design features.

Validation

Temporal holdout: trained on trials completing before 2020, tested on 133 post-2020 trials. AUC = 0.745, Brier = 0.197. See the Calibration tab for detailed metrics.

Limitations

  • Calibration slope below ideal range — predicted probabilities may be overconfident
  • AACT results data is incomplete for many trials (475/1259 unlabelable)
  • Drug class mapping relies on keyword matching, which may miss novel agents
  • Success is defined by the primary endpoint only — trials with positive secondary endpoints but negative primaries are labeled as failures
  • Some drug classes (e.g., steroidal MRA, ACEi, beta-blockers) have limited representation because AACT only covers post-2000 trials. Landmark pre-2000 trials (RALES, COPERNICUS, HOPE) are absent, causing historically inaccurate base rates for these classes. Predictions rely more heavily on design features and priors for underrepresented classes
